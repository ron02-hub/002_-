# マーケティング分析レポート設計書

## 📋 概要

本ドキュメントは、200名のアンケート回答者データを分析し、データアナリスト・マーケターとしての包括的なレポートをExcel形式で出力する設計書です。

**バージョン**: 1.0.0  
**作成日**: 2026-01-05  
**最終更新日**: 2026-01-05

---

## 🎯 レポートの目的

1. **データアナリストとしての分析**
   - 全分析手法による包括的なデータ分析
   - 統計的な傾向の特定
   - データの可視化

2. **マーケターとしての洞察**
   - ユーザーセグメントの特定
   - 購買意欲に影響する要因の分析
   - マーケティング戦略への提案

3. **小学生にもわかる説明**
   - 専門用語を避けた平易な表現
   - 具体例を用いた説明
   - 視覚的なグラフとチャート

---

## 📊 分析項目

### 1. 基本統計

#### 1.1 回答者属性
- 年齢グループ別分布
- 性別分布
- 都道府県別分布
- 運転歴の分布
- EV所有経験の有無
- 音への感度の分布

#### 1.2 回答状況
- 総回答者数
- 完了率
- 平均回答時間
- 各音声サンプルの評価数

### 2. SD法分析

#### 2.1 音声サンプル別の平均評価
- 8軸（静か、心地よい、高級感、先進的、力強い、安心、ワクワク、自然）の平均スコア
- 音声サンプル間の比較

#### 2.2 属性別の評価傾向
- 年齢グループ別の評価傾向
- 性別の評価傾向
- EV所有経験別の評価傾向

#### 2.3 可視化
- レーダーチャート（音声サンプル別）
- 棒グラフ（評価軸別）
- ヒートマップ（音声サンプル × 評価軸）

### 3. 購買意欲分析

#### 3.1 購買意欲の分布
- 音声サンプル別の購買意欲分布（1-7スケール）
- 平均購買意欲
- 購買意欲が高い（6-7）回答者の割合

#### 3.2 属性別の購買意欲
- 年齢グループ別の平均購買意欲
- 性別の平均購買意欲
- EV所有経験別の平均購買意欲

#### 3.3 可視化
- ヒストグラム（購買意欲の分布）
- 棒グラフ（音声サンプル別の平均購買意欲）
- 箱ひげ図（属性別の購買意欲）

### 4. 因子分析

#### 4.1 因子の抽出
- 主要因子の特定
- 各因子の説明力（寄与率）
- 因子負荷量

#### 4.2 因子の解釈
- 因子1: 「高級感・先進性」
- 因子2: 「静かさ・安心感」
- 因子3: 「力強さ・ワクワク感」
- （実際の分析結果に基づく）

#### 4.3 可視化
- 因子負荷量の棒グラフ
- スクリープロット（寄与率）
- 因子得点の散布図

### 5. クロス集計分析

#### 5.1 年齢 × 購買意欲
- 年齢グループ別の購買意欲分布
- 購買意欲が高い年齢層の特定

#### 5.2 性別 × 購買意欲
- 性別の購買意欲の違い

#### 5.3 EV所有経験 × 購買意欲
- EV所有経験の有無による購買意欲の違い

#### 5.4 可視化
- クロス集計テーブル
- 積み上げ棒グラフ
- 100%積み上げ棒グラフ

### 6. 評価グリッド法（価値ツリー分析）

#### 6.1 価値の階層構造
- 終極価値（Terminal Values）: 「なぜそれが重要か」の最終的な答え
- 手段価値（Instrumental Values）: 価値を実現するための方法
- 機能的属性（Functional Attributes）: 音の具体的な特徴（低音が響く、等）

#### 6.2 価値の頻度分析
- 抽出されたコンストラクト（評価基準）の集計
- ラダリング（上位・下位概念）で出現したキーワードのランキング
- 階層別の分布

#### 6.3 可視化
- 階層ツリーダイアグラム
- 価値の頻度ランキング
- 階層別の価値分布

### 7. NLP分析（自由記述）

#### 7.1 キーワード分析
- 出現頻度の高いキーワード
- キーワードの共起関係

#### 7.2 感情分析
- 平均感情スコア
- ポジティブ/ネガティブの分布

#### 7.3 可視化
- ワードクラウド
- キーワードランキング
- 感情スコアの分布

### 8. 最良・最悪音分析

#### 8.1 最良音の選択傾向
- 最も多く選ばれた音声サンプル
- 選択理由の分析

#### 8.2 最悪音の選択傾向
- 最も多く選ばれた音声サンプル
- 選択理由の分析

#### 8.3 可視化
- 最良音の選択率
- 最悪音の選択率
- 選択理由のワードクラウド

## 📝 マーケティング洞察の記述方針

### 小学生にもわかる説明の原則

1. **専門用語を避ける**
   - ❌ 「因子分析により、主成分が抽出されました」
   - ✅ 「200人の答えを調べたところ、3つの大きなグループに分けることができました」

2. **具体例を使う**
   - ❌ 「購買意欲が高い」
   - ✅ 「10人中7人が『この車を買いたい』と答えました」

3. **数字を分かりやすく**
   - ❌ 「平均購買意欲は4.2」
   - ✅ 「10点満点で、平均4.2点でした。これは『やや買いたい』という気持ちです」

4. **比喩を使う**
   - ❌ 「因子負荷量が高い」
   - ✅ 「この音は、『高級感』という特徴がとても強く出ています。まるで高級車のような印象です」

5. **ストーリー性を持たせる**
   - データの羅列ではなく、物語として説明
   - 「20代の若い人たちは...」「一方、50代以上の方は...」

## 🔍 分析の流れ

### Step 1: データ取得
1. Prismaから200名分のデータを取得
2. 回答者、評価、最良・最悪音、コンストラクト、インタビューログを結合

### Step 2: 基本統計の計算
1. 回答者属性の集計
2. 回答状況の集計

### Step 3: 各分析の実行
1. SD法分析（音声サンプル別、属性別）
2. 購買意欲分析（分布、属性別）
3. 因子分析（Pythonスクリプト実行）
4. クロス集計（年齢×購買意欲、性別×購買意欲など）
5. 価値ツリー分析（コンストラクトから階層構造を構築）
6. NLP分析（Pythonスクリプト実行）
7. 最良・最悪音分析（選択傾向と理由）

### Step 4: 可視化
1. 各分析結果をグラフ化
2. ExcelJSでグラフを作成

### Step 5: マーケティング洞察の作成
1. データから見える傾向を特定
2. ユーザーセグメントを定義
3. 購買意欲に影響する要因を分析
4. 小学生にもわかる説明で記述

### Step 6: 結論と提案
1. 主要な発見事項をまとめる
2. 結論を導く
3. 具体的な提案内容を作成

## 🎯 マーケティング洞察のテンプレート

### セグメント分析

```
## ユーザーセグメント1: 「高級感を求める層」

### 特徴
- 年齢: 40-50代が中心
- 購買意欲: 高い（平均5.5/7）
- 重視する点: 高級感、先進性

### 説明（小学生向け）
このグループの人たちは、お店で一番高い商品を選ぶような人たちです。
「高級感がある」「最新の技術が使われている」という音を好みます。
10人中8人が「この車を買いたい」と答えました。

### マーケティング提案
- 高級感を強調したプロモーション
- 先進技術の訴求
- 40-50代向けのマーケティングチャネル
```

### 購買意欲に影響する要因

```
## 購買意欲に影響する要因

### 要因1: 音の「静かさ」
- 影響度: 高い
- 説明: 音が静かなほど、購買意欲が高くなる傾向があります。
  静かな音を好む人は、10人中7人が「買いたい」と答えました。

### 要因2: 年齢
- 影響度: 中
- 説明: 20-30代の若い人たちは、50代以上の人たちより購買意欲が高い傾向があります。
  若い人たちは新しい技術に興味があるためです。

### 要因3: EV所有経験
- 影響度: 中
- 説明: すでにEVを持っている人は、持っていない人より購買意欲が高いです。
  EVの良さを知っているため、新しいEVにも興味があるのです。
```

## 📊 分析スクリプトの仕様

### 入力
- Prismaデータベース（200名分のデータ）

### 処理
1. データ取得と前処理
2. 各分析の実行
3. 統計計算
4. グラフ生成
5. Excel出力

### 出力
- Excelファイル（11シート）
- 各シートにデータテーブルとグラフ

### 実行方法

```bash
# データベースに200名分の擬似データを生成（初回のみ）
npm run db:mock:200

# マーケティング分析レポートを生成
npm run marketing:report
```
